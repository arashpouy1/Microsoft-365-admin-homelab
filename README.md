# Microsoft 365 Administration Homelab

Hands-on Microsoft 365 and Microsoft Entra administration projects developed within a realistic, scenario-based homelab environment.

This portfolio demonstrates practical experience with identity administration, user and group management, administrative access, authentication, security controls and Microsoft cloud administration.

---

## Lab Scenario — Bright Horizons Health

**Bright Horizons Health** is a fictional healthcare organisation created to provide a realistic business context for this Microsoft 365 administration lab.

The tenant includes users across multiple departments, job roles and business functions. User identities, organisational attributes, security groups and Microsoft 365 groups are progressively developed and reused across the projects.

The environment is designed to simulate practical administration scenarios rather than isolated technical exercises.

Example organisational areas include:

- Clinical
- Reception
- Finance
- Human Resources
- Information Technology
- Procurement
- Management

## Progressive Lab Approach

This portfolio is designed as a connected administration environment rather than a collection of unrelated exercises.

**Users and organisational attributes → Groups and dynamic membership → Administrative roles and RBAC → Authentication and Security Defaults → Conditional Access → Enterprise Applications and App Registrations**

Each project extends the existing environment and introduces additional Microsoft 365 and Microsoft Entra administration capabilities.

For example, users and organisational attributes created during Project 01 were reused during Project 02 to automate group membership based on Department and Job Title. Groups created during Project 02 will support role assignment, access control and security scenarios in later projects.
